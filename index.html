About Network Control Tool

The Network Control Tool is a sophisticated GUI application developed as part of the ORDNANCE FACTORY DEHRADUN (OFD) summer training program in 2024. This tool is designed to facilitate the monitoring and management of network devices using their IP addresses. It offers a comprehensive set of features aimed at optimizing network diagnostics and management tasks.

    Features:

  • Automated Diagnostics:
    Run Diagnostics: Initiates a network scan to check the connectivity status of registered IP addresses. The tool displays which devices are disconnected or experiencing issues.

  • IP Address Management:
    Add IP Address: Allows users to input new IP addresses along with associated details such as priority, location, and remarks. These entries are stored in a structured SQLite database.

  • Modify IP Address:
    Provides functionalities to edit or delete existing IP address entries. Users can update details such as IP address, priority, location, and remarks directly from the GUI.

  • Interval Management:
    Set Time Interval: Enables users to specify the interval (in seconds) at which the network scans and diagnostics should automatically repeat. This ensures continuous monitoring without constant user intervention.

  • User-friendly Interface:
    Responsive Design: Implemented using Tkinter library in Python, ensuring a user-friendly and intuitive graphical interface. It includes features like a gradient background, customizable navbar, and pop-up dialogs for alerts and notifications.

  • Developer's Corner:
    Highlights the contributions of the development team, including responsibilities and technologies used by each member. This section acknowledges the effort and expertise put into creating and testing the Network Control Tool.

How It Works:

The Network Control Tool utilizes a combination of backend logic and database operations to manage IP addresses and their associated details. It employs threading for concurrent operations, ensuring that the GUI remains responsive during network scans and database transactions. The SQLite database (`ip_address.db`) stores all IP address entries securely and facilitates efficient data retrieval and modification.

Instructions:

  • Run Diagnostics - starts network scan and prompts the disconnected nodes. It will repeat the diagnostics of all network after mentioned time interval.
  • Add IP Address - to input new IP addresses, priority, location, and remarks.
  • Modify IP Address - to edit or delete existing IP entries.
  • Terminate Diagnostics - to stop ongoing diagnostics.
  • Set Time Interval - to specify automatic pinging intervals (in seconds).

Conclusion:

The Network Control Tool is a testament to effective collaboration and innovation in network management. Developed by a team of dedicated interns during the OFD summer training program, it represents a practical solution for automating network diagnostics and enhancing operational efficiency. Whether used in educational settings, research institutions, or industrial environments, the tool stands ready to support robust network monitoring and management needs.
